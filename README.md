## ðŸ“– Deskripsi Proyek
**ADD** adalah proyek sosial (*Social Project*) yang bertujuan untuk meningkatkan literasi digital dan menyediakan alat pertahanan siber bagi masyarakat umum. Proyek ini menggabungkan:
1.  **Kecerdasan Buatan (AI):** Menggunakan *Deep Learning* (MobileNetV2) untuk membedakan wajah asli dan *Deepfake*.
2.  **Kriptografi & Forensik:** Menerapkan Hashing (SHA-256) untuk integritas data dan Enkripsi (AES) untuk privasi pelapor.
3.  **Partisipasi Publik (Crowdsourcing):** Wadah bagi masyarakat untuk berkontribusi data guna melatih model AI agar semakin pintar.

## ðŸš€ Cara Memulai (Quick Start)

1.  **Clone Repositori:**
    ```bash
    git clone [https://github.com/Garmandsk/aplikasi-deteksi-deepfake](https://github.com/Garmandsk/aplikasi-deteksi-deepfake)
    cd aplikasi-deteksi-deepfake
    ```

2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Jalankan Aplikasi:**
    Masuk ke folder aplikasi dan jalankan Streamlit.
    ```bash
    cd app_files
    streamlit run app.py
    ```
